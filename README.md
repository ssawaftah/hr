# نظام إدارة مصنع الحديد

تطبيق ويب ثابت لإدارة مصنع حديد كبير، يحتوي على واجهة موظف وواجهة مسؤول، ويرتبط بـ Firebase Realtime Database و Firebase Storage.

## التشغيل

افتح `index.html` مباشرة أو شغل خادم محلي من هذا المجلد:

```powershell
python -m http.server 5173
```

ثم افتح:

```text
http://localhost:5173
```

## بيانات الدخول التجريبية

- الموظف: `1001` وكلمة السر `1234`
- المسؤول: `9001` وكلمة السر `admin`

عند أول تشغيل سيحاول التطبيق إنشاء بيانات تجريبية في:

- `employees`
- `admins`
- `announcements`
- `requests`

## ملاحظات Firebase

هذا النموذج يستخدم رقم الموظف وكلمة سر وهمية كما طلبت. للإنتاج الفعلي يفضل الانتقال إلى Firebase Authentication أو Cloud Functions حتى لا تُحفظ كلمات السر كنص ظاهر في قاعدة البيانات.
